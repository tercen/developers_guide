Tercen app library

The Tercen app library is managed by a json file. Currently Tercen support is the only authorized to modify it.

Anyone can request an app be added to the Tercen app library.

Here are the steps for adding it to Tercen app library:

  • Notify Tercen support (support@tercen.com) regarding your wish to add it to app-library
  • Tercen support tests the app
  • Tercen support modifies the app-library repository (publish)
  • Tercen support send email to indicate it is available
